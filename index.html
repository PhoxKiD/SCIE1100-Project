Welcome to Kryptos

The purpose of Kryptos is to teach anyone the basics of Cryptology, along with some of the early applications of Cryptography, such as the Caesar cipher and the Vingenere cipher.

Encryption is a modern application of cryptology. Encrypting something means to alter it in another way using a key, so it becomes unreadable. It can be decrypted using the same key. Today, encryption is used to keep Internet connections secure and and files safe.


Canvas

This javascript applet takes the input of letters and numbers to form an arrangement of colours on the canvas, which can be resized using the draggable bar.

Instructions: Use the draggable bar to pick a size for the canvas, then type a message into the box right below it and press enter. You will see your message as colours fill up the box from left to right, top to bottom.


Converter

This javascript applet takes a string of letters and numbers and outputs a sequence of colours inside the blue box. This sequence can be copied into an email message.

Instructions: Enter a sequence of numbers and letters and press enter. You will see a sequence of colours below. Select the text and copy it. You can paste it into an email message.
